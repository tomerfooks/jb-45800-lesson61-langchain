# UI/UX Pro Max Skill

Use this skill when improving the design of the frontend.

## Goals
- Make the interface feel modern, polished, and intentional.
- Improve spacing, typography, hierarchy, and visual consistency.
- Add subtle depth: gradients, shadows, borders, blur, and motion.
- Ensure responsive behavior on mobile and desktop.
- Preserve usability and accessibility.

## Design Principles
- Prefer a strong visual hierarchy.
- Use a limited, cohesive color palette.
- Increase whitespace and alignment consistency.
- Make primary actions obvious.
- Use rounded corners, soft shadows, and tasteful glassmorphism where appropriate.
- Avoid clutter and overly dense layouts.

## Implementation Checklist
- Update layout structure for better balance.
- Improve header/hero sections with clearer branding.
- Upgrade cards, buttons, inputs, and badges.
- Add hover/focus states and transitions.
- Ensure readable contrast.
- Make mobile layout clean and full-width.
- Keep interactions simple and fast.

## Output Expectations
When asked to improve UI, modify the frontend directly and aim for a premium, production-ready look.
